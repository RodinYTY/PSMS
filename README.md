## 功能

1. 教师、学生、课程、琴房、琴房内乐器的信息增、删、改、模糊查找
2. 修改数据有撤回功能，提升用户操作灵活性
4. 用户注册和登录，三种角色三种界面
5. 教师邀请码机制，只能由管理员生成，有效期为7天，触发器自动删除
6. 师生的上课记录统计，老师可以签退
7. 个人信息的修改
